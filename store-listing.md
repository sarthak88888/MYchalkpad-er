# MyChalkPad — Store Listing Content

## APP NAME
MyChalkPad - School ERP

## SHORT DESCRIPTION (80 characters max — for Google Play)
Complete school management: attendance, fees, SMS alerts & bus tracking.

## FULL DESCRIPTION (4000 characters max — for Google Play)
MyChalkPad is a complete School ERP (Enterprise Resource Planning) system
designed specifically for Indian schools. Manage your entire school from
one powerful, easy-to-use mobile app.

🏫 FOR SCHOOL ADMINISTRATORS & PRINCIPALS
- Dashboard with real-time stats: students, staff, attendance, SMS sent
- Admissions management: approve or reject applications instantly
- Fee management: track payments, mark dues, send reminders
- Staff management: salaries, roles, contact information
- Timetable editor: create and update class schedules
- Transfer certificates: generate and track TCs automatically
- UDISE data export: stay government-compliant with one tap
- Inspection preparation: checklist with readiness percentage
- Dropout tracking: monitor and reduce student dropout rates
- Student rankings: class-wise, subject-wise performance rankings

📊 FOR TEACHERS
- Mark attendance in seconds with Present/Absent/Late buttons
- Automatic SMS and WhatsApp alerts to parents of absent students
- View class timetable and student performance

👨‍👩‍👧 FOR PARENTS
- Real-time attendance notifications via SMS and WhatsApp
- Live bus GPS tracking — know exactly where the school bus is
- Fee payment via UPI or Razorpay online payment
- View child's performance, marks and attendance history
- Submit and track complaints directly to school administration

🚌 FOR BUS DRIVERS
- GPS route tracking with one tap start/stop
- Student boarding management
- Route information at a glance

💬 COMMUNICATION
- Bulk SMS in Hindi, Punjabi, Kangri, Haryanvi and English
- WhatsApp alerts via Interakt
- PTM meeting scheduling with automated reminders
- Complaint management with status tracking

🔒 SECURITY
- Firebase Phone OTP authentication — no passwords needed
- Fingerprint/Face ID login for quick access
- Role-based access: each user sees only their own screens
- Bank-grade Firebase security rules

📱 MADE FOR INDIA
- Works on 2G/3G connections
- Hindi and English interface
- UPI payment support
- Indian phone number authentication
